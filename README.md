# AI Assistant Matchmaker

A simple, interactive web app that helps users find the right AI assistant for their needs through a decision tree questionnaire.

## How It Works

Answer a few quick questions about your workflow, and the app will recommend the AI assistant that best fits your needs. The decision tree considers factors like:

- Your primary use case (research, creative work, coding)
- Document handling requirements
- Privacy concerns
- Ecosystem preferences (Google, Microsoft, open-source)
- Desired features (code execution, citations, real-time data)

## Tech Stack

- Pure HTML, CSS, and JavaScript (no frameworks)
- DM Sans font via Google Fonts
- Responsive design for mobile and desktop

## Local Development

Simply open `index.html` in a web browser. No build process or dependencies required.

## Deployment

This project is designed for GitHub Pages. To deploy:

1. Push to a GitHub repository
2. Enable GitHub Pages in repository settings
3. Select the main branch as the source
